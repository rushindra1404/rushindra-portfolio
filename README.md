# Bandaru Rushindra - Creative AI/ML Engineer Portfolio 🚀

Welcome to my personal portfolio repository! This project exhibits my developer journey, technical skillsets, and creative milestones in Artificial Intelligence, Machine Learning, Generative AI, Cloud Deployments, and Futuristic UI/UX digital design.

---

## 🛠️ Core Technology Stack

- **AI & Machine Learning**: Python, PyTorch, TensorFlow, Scikit-Learn, Pandas, NumPy, Natural Language Processing (NLP), Computer Vision
- **Generative AI & Agent Architectures**: OpenAI APIs, LangChain, LlamaIndex, Autonomous Multi-Agent Workflows (CrewAI), Vector Databases, Prompt Engineering
- **Immersive 3D Web Experiences**: Three.js, WebGL, GLSL Shaders, React Three Fiber (R3F), GSAP ScrollTrigger Animations, Framer Motion
- **Frontend & Infrastructure**: React, TypeScript, Vite, Node.js, FastAPI, Docker, Kubernetes (GKE), Amazon Web Services (AWS)

---

## 📂 Key Architecture highlights

- **3D Spatial WebGL Canvas**: Implements a Draco-compressed custom 3D model utilizing Three.js and custom camera orbit controls to create a reactive, native desktop and stacked mobile experience.
- **Searchable Certifications Portal**: Built using **React Portals** (`createPortal`) to guarantee seamless overlay mounting directly on `document.body` outside of parent GSAP scroll transforms. Features real-time query filtering and custom SVG dynamic file previews.
- **GSAP Scroll-Stacking Cards**: A cinematic projects stacking sequence utilizing ScrollTrigger timelines to glide and dissolve cards based on scrolling velocity.
- **Native Fullscreen Navigation Overlay**: A modern, touch-friendly viewport menu supporting lock scroll body hooks and stagger layout animations.

## ⚙️ Development Instructions

### Local Environment Setup
Install dependencies:
```bash
npm install
```

Start the Vite local development server:
```bash
npm run dev
```

Build the optimized static assets for production:
```bash
npm run build
```
